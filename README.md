# OSX CPU Temp 1.0.0

Outputs current CPU temperature in °C for OSX.

## Usage

### Compiling
```
make
```

### Running

```
./sensors
```

or

```
sudo make install # installs to /usr/local/bin
sensors
```

### Output example

```
temp: +64.1°C
```
